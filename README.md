# Radium Launcher (Unofficial) 🟢
An unofficial, open-source launcher for Radium (the Rec Room revival). This program was created to solve the clunky experience of digging through file directories just to switch between VR and Screen Mode (and to just launch the game).

# NOTICE: Run setup.exe, don't bother running the .msi

## 🚀 Features

- One-Click Launch: Easily switch between Screen Mode and VR Mode without using batch files.
- Auto-Detection: Automatically finds your Radium directory. If you move your game files, the launcher will prompt you to relink them.
- Process Manager: Built-in "Kill Process" button to quickly close the game if it freezes or hangs in the background.
- Live News Feed: Stay updated with the latest community photos and announcements directly from the official Radium website.
- Discord Integration: Quick link to join the community and get support.

## 🛠️ Installation

1. Download the latest installer from the Releases page.
2. Run the setup and open the launcher.
3. On the first launch, select the folder where your Radium.exe is located.
4. Hit Launch and enjoy!

## 🔒 Security & Transparency Because this is a community-made tool, I have made the source code 100% public.

- No Data Collection: This app does not have access to your Radium account info or APIs.
- Open Source: Feel free to look through the C# code to see exactly how it handles your files.
